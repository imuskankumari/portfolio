HyperText Markup Language (HTML)

What is HTML ?


HTML stands for HyperText Markup Language. It is the standard code used to create and structure everything you see on a website.If a website were a house, HTML would be the skeleton and the walls. It doesn't handle the "paint" (that’s CSS) or the "electricity/plumbing" (that’s JavaScript), but it defines where the rooms are and what they contain.
